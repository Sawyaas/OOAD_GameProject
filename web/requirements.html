1) User must be able to create a character. This character’s name will between 6 and 12 characters long. This Character will automatically become the “Player”

2) User will be able to select location for base. When location is selected no other (AI) bases can populate there.

3) Player will initially be given a default value of stock (food, wood, stone)

4) Each army the user creates will be assigned to the user’s base.

5) The user will be able to build units for their army once they have created the required buildings and have met the required stock.

6) The user will be able to create building once they have the required wood and stone values.

7) The user can select where in the base they wish to place each building they create.

8) When a building is destroyed within the players base, the cell is made available and 50% of the resources are returned to the player.

9) When the player successfully attacks and defeats another base, a random amount of the losing bases’ resources and surviving units are added to the player’s base.

10) When the player attacks a base, or is being attacked, the total defence statistics of both bases are used to determine the winner.

11) If the player’s base is completely destroyed, the game ends and the player has lost.

12) If the player destroys all enemy bases on the map, the game ends and the player has won.
