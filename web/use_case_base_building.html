Fully Dressed Use Cases (1/2)

Use Case Id: 2.0
Use Case Title: Base Building
Primary Actor: Player
Secondary Actor: System
System Under Discussion: Real Time Strategy Game “Last Whisper”
Level: Player Goal
Pre-conditions: Player is in-game and has created a character.
Post-conditions: The Player’s base has been updated, buildings and army units add to the base’s defence value. The players stock has been reduced to reflect the player’s base edit.
Basic Flow:
  • 1.
    Player selects “Build Base” option.
  • 2.
    Player selects base type - Premade base or Empty base.
  • 2.1.
    By selecting a premade base the player is given key start off buildings in their base such as a barracks, storage shed and farm. Players wood and stone stock will be reduced straight away.
  • 2.2.
    By selecting the empty base the player is given an empty lot with basic walls.
  • 3.
    Player drags and drops base onto a section of the game map.
  • 3.1.
    The lot the player chooses cannot be preoccupied by any other base.
  • 4.
    Player enters Edit Mode.
  • 5.
    System displays Base in grid format.
  • 6.
    Player selects edit option - Building Edit and Unit Edit.
  • 6.1.
    A sub-option of “Defence Edit” will be made available when a Lab is built on base. This will allow Siege Equipment and Ballistas to be added to the base.
  • 7.
    Player enters Building Edit.
  • 8.
    System displays a selection of buildings available to the player dependent on the player’s wood and stone count.
  • 8.1.
    The buildings available shall be Barracks, Fighting Range, Archery Range, Kennels, Stables, Labs, Food Storage, Farms, and Walls.
  • 9.
    Player uses drag and drop to place a building in their base.
  • 9.1.
    System determines if cell is empty.
  • 10.
    Player’s wood and stone count reduces with each building built.
  • 11.
    Player returns to Edit Mode.
  • 12.
    Player enters Unit Edit.
  • 12.1.
    Player must have a Barrack on their base in order to access this function.
  • 13.
    System presents player with their current army and a number of empty unit fields.
  • 13.1.
    The Unit Fields available will include Swordsmen, Spearmen, Archers, Dogs and Cavalry.
  • 14.
    Player delegates a number of their soldiers to a unit increasing their defence value.
  • 14.1.
    Player may access these units only if they have the required building on base.
  • 14.2.
    Archers require an Archery Range, Dogs require a Kennel, Cavalry require Stables etc.
  • 15.
    Player returns to Edit Mode and saves.
  • 16.
    Use case ends when Player commits to save and exits to game map.
