Brief Use Cases

UC_4.0: Destroy a Building

Allows the player to destroy the building within his base to free up a cell.

UC_7.0: Check Information About a Building

Allow the Player to check information about a building before creating it or about a building already built on his base.

UC_8.0: Check Information About an Unit

Allow the Player to check information about a unit before creating it or about a unit already created on his army.

UC_9.0: Check Information About His Own Base

Allow the Player to check global information about his base.

UC_10.0: Check Information About Another Base

Allow the Player to check information about an enemy base on the map.
