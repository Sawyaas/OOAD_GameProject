Use Case 6.0: Attack Another Base

Primary Actor: Player
System under discussion: Turn Based Strategy Game "Last Whisper"
Level: Player Goal
Preconditions: The player is logged on, has started a new game or loaded a previous one and is on his base map
Basic Flow:
  1. Player goes to the map
  2. Player selects the target base
  3. Player selects the Attack option
  4. System calculates the total Attack value of the Playerâ€™s army as the offensive side and the total Defence value of the target base as the defensive side
  5. System determines the winner by comparing the Attack value of the offensive side and the Defence value of the defensive side
    1. If the Attack value of the offensive side is at least 50% higher than the Defence value of the defensive side then the offensive side automatically wins
  6. System gives a random part of the resources and a random part of the remaining units of the winner side to the looser side
  7. System shows the final battle screen
  8. Player comes back to the updated map
Alternative Flows:
  • 5.1.a. If the Defence value of the defensive side is at least 50% higher than the Attack value of the offensive side then the defensive side automatically wins.
  • 5.1.b. If the Attack value of the offensive side is equals to the Defence value of the defensive side then System randomly determines the winner
  • 6.1 If the defensive side has lost then System sets the attacked base as destroyed
Extensions:
    1. Player check information about a base
    2. Player comes back to the map
Postconditions: The armies and resources of the both sides are updated. If Player has won, the target base is destroyed and the map is updated.
