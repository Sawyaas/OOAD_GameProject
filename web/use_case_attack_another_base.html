Use Case 6.0: Attack Another Base

Primary Actor: Player
System under discussion: Turn Based Strategy Game "Last Whisper"
Level: Player Goal
Preconditions: The player is logged on, has started a new game or loaded a previous one and is on his base map
Basic Flow:
  1. Player goes to the map by clicking on the Map button at the top-right of the interface
  2. Player selects the target base by clicking on it on the map
  3. Player selects the Attack option
  4. System checks the total Attack value of the Playerâ€™s army and the total Defence value of the target base
  5. System determines the winner. The Attack value of the offensive side is 50% higher than the defensive side. Player win
  6. A random part of the resources of the target base is given to the Player. A random part of the remaining units of the loser army is given to the Player
  7. The target base is destroyed
  8. System shows the final battle screen
  9. Player come back to the updated map by closing the battle screen
Alternative Flows:
  • 5.a. The Defence value of the defensive side is 50% higher than the Attack value of the offensive side. Player loose.
  • 5.b. The Attack value of the offensive side is equals to the Defence value of the defensive side
    1. System randomly determines the winner
    2. Player win
    3. Resume the Basic Flow at step 6
  • 5.c The Attack value of the offensive side is equals to the Defence value of the defensive side
    1. System randomly determines the winner
    2. Player loose
    3. Continue to the Alternative Flow 6.a
  • 6.a. A random part of the resources of the Player is given to the attacked enemy. A random part of the remaining units of the Player army is given to the attacked enemy.
    Skip step 7 and resume the Basic Flow at step 8
Extensions:
    1. Player check information about a base by clicking on it on the map
    2. Player come back to the map by closing the information window
    3. Resume to Basic Flow at step 2
Postconditions: The armies and resources of the both sides are updated. If Player has won, the target base is destroyed and the map is updated.
